# Chapter 3 数据链路层

## 本章的主要结构：

- 数据链路层设计相关问题
  - 为网络层提供服务
  - 解决传输中的错误
  - 控制流量 (Flow-control)

- 错误检测与校正
  - 纠错码 (error-Correcting Codes)
  - 检错码 (error-Detecting Codes)
- 基本的数据链路层协议
  - 一种乌托邦式的单工 (simplex) 协议
  - 无错误信道上的单工的停-等 (stop-and-wait) 协议
  - 有错误信道上的单工的停-等 (stop-and-wait) 协议
- 滑动窗口协议(Sliding Window Protocols)
  - 1位的滑动窗口协议
  - 使用回退N帧方法的滑动窗口协议
  - 使用选择性重复的滑动窗口协议
- 数据链路层的例子
  - SONET
  - ADSL
